Git Hub Docs

CoVE

CoVE is an web application to Convert, Validate and Explore data following certain open data standards - currently 360Giving and the Open Contracting Data standard. http://cove.opendataservices.coop

Open Aid Geocoder

This tools allow users to attach geographical information to existing aid projects.

D-Portal

D-portal.org helps you explore IATI data by providing information by country or by publisher on development activities and budgets published to the platform.

OIPA

OIPA extracts all published IATI XML files from the IATI Registry and stores them in a PostgreSQL database, that you can access using a RESTful API. https://www.oipa.nl
